# Intelligent Failure Pattern Detection

A machine learning based log analysis system that detects failure patterns from application logs.

## Features
- Log file analysis
- Failure pattern detection
- User authentication
- ML-based prediction

## Files
- app.py : Main application
- auth.py : Authentication module
- model_utils.py : ML utilities
- sample_logs.csv : Sample dataset
- users.json : User credentials
- requirements.txt : Dependencies

## Installation

pip install -r requirements.txt

## Run

python app.py
